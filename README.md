### What I Build

#### 1. Multilingual Call Center Automation
Deploying voice-first systems using **Retell AI** and **ElevenLabs** that handle live customer support calls. These agents dynamically detect the language mid-call, fetch user data from internal databases in real time, and perform warm transfers to human agents when necessary.

#### 2. Autonomous Task Resolution & Routing
Building agents that listen to **ClickUp** webhooks, classify incoming tickets using custom models, and either resolve the issue automatically or route it to the correct department with a generated brief.

#### 3. Voice-to-Action Pipelines
Utilizing **n8n** to ingest audio recordings (voicemails or meeting notes), transcribing them via open-source Whisper models, extracting actionable items using an LLM, and automatically populating structured databases and task boards.

#### 4. Custom AI Endpoints
Wrapping fine-tuned models (both text and vision-language) in **FastAPI** backends and exposing them as secure, scalable APIs for internal enterprise tools.
